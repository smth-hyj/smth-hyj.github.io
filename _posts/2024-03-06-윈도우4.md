## 장애 조치 클러스터

1. [ex] NAS 설치  
    + 준비 사항
        - CD/DVD: FreeNAS-11.3.U2.iso
        - VM 구성
            - Guest OS: [x] OTHER
            - Version FREEBSD 최신
            - VM name FREE NAS
            - Disk Size : 20G
                - 추가 디스크 3장 더 장착(10g * 3)
    + NAS 설치
        - Console SEtup : 1. Install/Upgrade
        - Choose Destination media : [x] da0 (20g)
        - Free NAS installation : yes
        - root 암호 : soldesk1.
        - FREE NAS BOOT mode : boot via bios
2. [ex] AD 구성 - FIRST  
    1. 역할 및 기능 추가
        - Active Directory Domain SErvices
    2. 서비스 기동(x)
    3. 서비스 설정(dcp)
3. [ex] FailOver Cluster 구성 - SECOND, THIRD  
    1. 역할 및 기능 추가
        - 기능 추가 ㅣ 장애 조치 클러스터링
    2. 서비스 기동(X)
    3. 서비스 설정
        - 유효성 검사
        - 장애 조치 클러스터 구성
4. [ex] 테스트 - FIRST
