## DHCP 서버 
### DHCP 서버 동작
- DORA 과정(DISCOVERY, OFFER, REQUEST, ACK)
- DHCP 서버와 DCHP 클라이언트는 일반적으로 같은 대역에 있어야한다. => BOOT Relay Server
- 같은 대역에는 DHCP 서버가 일반적인 경우 1대만 존재해야 한다. => 부하분산

[참고] 고정 IP vs 동적 IP  
[참고] 공인 IP vs 사설 IP
